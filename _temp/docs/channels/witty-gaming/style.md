# Style Guide — Witty Gaming

## Format

- Super super-cuts — very tight, only the best moments
- Voiceover included — pitched/processed for comedic effect

## Tone

- Funny, punchy, fast-paced
- Voice style TBD (pitched, cartoonish — experimenting)

## Editing

- Heaviest editing of the three channels
- Voice pitch/processing applied to commentary
- Cut aggressively — only funniest moments survive
